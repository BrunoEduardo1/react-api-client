# Busca de séries

Cliente API com reactjs. Busca de séries com a api tvmaze.

## Quick Start

```bash
# Instalar depêndencias
npm install

# Excuta o app em modo de desenvolvimento
npm start

# Abra [http://localhost:3000](http://localhost:3000) para vê-lo no browser

```
